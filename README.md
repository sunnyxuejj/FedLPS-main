# FedLPS
The pytorch version for FedLPS

## Datasets
- MNIST
- CIFAR10
- CIFAR100
- Tiny-Imagenet ([FedLPS_AE](https://zenodo.org/records/10811717))
- Reddit (in data/reddit/)

## Running FedLPS
```
cd /script/
sh cifar100_FedLPS.sh
```

## Running Baselines
```
cd /script_baselines/FedAvg/
sh cifar100_FedAvg.sh
```
